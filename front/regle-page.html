Les règles de Kick'Oridor

  • Jeu à deux joueurs seulement, en mode 1 contre 1.
  • Plateau de jeu de dimension 9x9.
  • Chaque joueur commence sur une ligne différente, avec une visibilité spécifique pour chaque case.
  • Les joueurs peuvent placer des murs de deux cases de longueur pour bloquer leurs adversaires.
  • La visibilité des cases est modifiée en fonction des murs placés par les joueurs.
  • Les joueurs peuvent uniquement voir les cases avec une visibilité de 0 ou plus (pour le joueur A) ou de 0 ou moins (pour le joueur B).
  • L'objectif est d'atteindre la ligne opposée avant l'adversaire.
  • Les joueurs peuvent placer des murs ou se déplacer d'une case dans une direction cardinale à chaque tour.
  • Les joueurs ne peuvent pas sauter par-dessus les murs mais peuvent sauter par-dessus les autres joueurs.
  • La partie se termine en cas de victoire d'un joueur, de match nul après 100 tours de chaque joueur, ou si un joueur atteint sa ligne de but avant l'adversaire.

Comment Jouer

  • Le jeu se joue avec la souris
  • Cliquez sur votre joueur pour afficher les déplacements possibles
  • Cliquez sur une des cases affichées pour déplacer votre joueur
  • Cliquez sur une intersection de cases pour prévisualiser le placement d'un mur
  • Cliquez au centre sur le mur pour effectuer une rotation de 90°
  • Valider le mur pour le placer
